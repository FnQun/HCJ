##### JSON基本 
* JavaScript Object Notaion
```
<!--js-->
{
name: "jerry",
age: 1
}

---

<!--JSON-->
{
"name": "jerry",
"age": 1
}
```
> JS- - - - -1. 不会报错  
> JSON- - 1. 会报错
